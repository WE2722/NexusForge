# NexusForge

Multi-agent orchestration system for AI & Dev projects.
Python 3.13 Backend + React 19 Frontend.

## Setup

1. Copy `backend/.env.example` to `backend/.env` and fill in your API keys
2. Backend: `cd backend && pip install -r requirements.txt && uvicorn app.main:app --reload`
3. Frontend: `cd frontend && npm install && npm run dev`
